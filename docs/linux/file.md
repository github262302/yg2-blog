# 文件操作

## 创建 编辑 查看
```
touch 123.py    // 当前目录创建123.py文件
mkdir 123       // 创建名为123目录
vi 123.py       // 编辑123.py
cat 123.py      // 命令行输出123.py的内容
```
- [vi具体操作](https://www.runoob.com/linux/linux-vim.html)
## 移动文件或文件夹
```
mv {{源文件}} {{目标目录}}
```

## 删除操作

```
rm 1.txt    // 删除单个文件
rm -r file  // 删除file目录
rm -rf file // 删除file目录(无需确认)
```
